Bandit Level 1 → Level 2

Level Goal

http://overthewire.org/wargames/bandit/bandit2.html

The password for the next level is stored in a file called - located in the home directory

Steps

  • Use ls command to find the file
  • As the file is named -(hyphen) we won’t be able to read it simply by cat command. As cat command considers -(hyphen) as stdin/Stout.
  • So prefix the command with the path ./
  • Use command : cat ./-

Commands Used

  • ls
  • cat ./-
