The password for the next level is stored in a file called spaces in this filename located in the home directory

Steps

    Method 1

  • If we directly use cat command, it won’t be able to find the file.
  • Since Cat command considers space as null and reads file name only until space
  • So use command : cat spaces\ in\ this\ filename

      • Method 2

    • We will write the name of the file in quotes, this will help us to read the password stored as shown in the given figure.
    • Use command : cat ‘spaces with filename’

    Commands Used

    • ls
    • cat spaces\ in\ this\ filename
    • cat ‘spaces with filename’
